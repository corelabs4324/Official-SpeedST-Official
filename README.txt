SPEED·ST Website — GitHub Upload Package
==========================================

Drag ALL files in this folder into your new GitHub repo root
(not into a subfolder — index.html must sit at the repo root).

Before uploading, ensure assets/ has all PNGs:
  python3 download-assets.py
  (or double-click PREPARE-UPLOAD.command)

Enable GitHub Pages:
  Settings → Pages → Deploy from branch: main / root

Your live URL will be:
  https://YOUR_USERNAME.github.io/REPO_NAME/

Optional: Stripe checkout requires deploying api/ to Vercel — see STRIPE-SETUP.md.
